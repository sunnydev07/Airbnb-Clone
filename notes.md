# Project Notes: Airbnb Clone

## Overview
Full-stack Airbnb clone built with Node.js/Express and MongoDB.

## Tech Stack
- **Runtime:** Node.js
- **Web Framework:** Express.js
- **Database:** MongoDB (via Mongoose)
- **View Engine:** EJS (using ejs-mate)
- **Authentication:** Passport.js

## Components
- **Models:** Data definitions (Listings, Users, Reviews)
- **Controllers:** Request logic
- **Routes:** API/page endpoints
- **Middleware:** Security, sessions, request handling
- **Views:** Server-side templates
- **Validation:** Joi (schema validation)
